Remarks

For methods that return byte arrays or that write into spans of bytes, this implementation emits the answer in the Big Endian byte order so that the CRC residue relationship (CRC(message concat CRC(message))) is a fixed value) holds. For CRC-64 this stable output is the byte sequence { 0x00, 0x00, 0x00, 0x00, 0x00, 0x00, 0x00, 0x00 }.

There are multiple, incompatible, definitions of a 64-bit cyclic redundancy check (CRC) algorithm. When interoperating with another system, ensure that you are using the same definition. The definition used by this implementation is not compatible with the cyclic redundancy check described in ISO 3309.
